# XSiteGen

#### [English](README.md) &bull; [Russian](README-RU.md)

</div>

**XSiteGen**&nbsp;&mdash; минималистичный генератор статических сайтов, разработанный с упором на простоту и удобство использования, в том числе для пользователей, не желающих тратить время и нервы на освоение и настройку более сложных генераторов.

XSiteGen был написан мной для моих личных потребностей, а также для изучения основных принципов работы генераторов статических сайтов. Тем не менее, я буду искренне рад, если моя работа кому-либо пригодится и будет интересна.

## Установка

Для работы требуется Python версии 3.9 или новее и модуль [Python-Markdown](https://pypi.org/project/Markdown/).

## Использование

Загрузите XSiteGen (клонируйте репозиторий или [загрузите последнюю версию](https://github.com/ivan-movchan/xsitegen/releases/latest)) и запустите модуль `xsitegen.py`.
Запуск модуля `xsitegen.py` с флагом `-v` или `--version` выведет используемую версию скрипта.

Первая строка каждого исходного файла должна содержать заголовок веб-страницы. С третьей строки должно начинаться содержимое веб-страницы (текст в формате Markdown).

XSiteGen способен загружать конфигурационный модуль из любой директории, отличной от той, в которой расположен сам XSiteGen. Вы можете скопировать модуль `config.py` в папку с исходными файлами собираемого сайта, отредактировать его, а затем запустить XSiteGen в этой директории, чтобы собрать сайт. Сам XSiteGen должен быть доступен по "путям" (переменная среды окружения `PATH`).

В каталоге `demo` Вы найдёте образец статического сайта, который может быть сгенерирован с использованием XSiteGen.

> [!WARNING]
> Python-Markdown является интерпретацией *John Gruber's Markdown*, но не более распространённого стандарта *CommonMark*. Возможно, Вам потребуется отредактировать исходные файлы для лучшего преобразования в HTML.
