### XSiteGen 2.0 — 2026-02-01

- **Добавлено:**  переменные директорий (папок) и страниц.
- **Добавлено:**  список приоритета переменных.
- **Добавлено:**  копирование файлов из исходных каталогов.
- **Добавлено:**  поддержка расширений Python-Markdown.
- **Добавлено:**  текстовые отступы (применяются только к тексту исходных файлов).
- **Добавлено:**  проверка версии Python при запуске.
- **Добавлено:**  вывод версий Python и Python-Markdown при запуске.
- **Добавлено:**  вывод времени работы XSiteGen по завершении.
- **Добавлено:**  флаг `-v` для ведения программой подробного журнала.
- **Обновлено:**  Python 3.10 требуется для запуска XSiteGen. Поддержка Python 3.9 была прекращена библиотекой Python-Markdown.
- **Обновлено:**  старый флаг `-v` (вывод версии программы) заменён на `-V`.
- **Обновлено:**  документация — теперь в папке `docs`.
- **Обновлено:**  логотип.
- **Обновлено:**  сообщения журнала.
- **Обновлено:**  обработка ошибок загрузки конфигурационного модуля.
  
  Если модуль нашёлся, но в нём есть ошибка, появится краткое сообщение с описанием ошибки.
  Если модуля нет вовсе, пользователю будет предложено создать новый конфигурационный модуль в текущей папке, либо перейти в другую папку и запустить XSiteGen в ней.

- **Исправлено:** страницы генерировались из шаблонов без `{content}`.
- **Удалено:**    бесполезные маркеры в начале каждого сообщения.
- **Удалено:**    "специальные" переменные.
- **Удалено:**    необходимость прописывать в конфигурационном модуле все поддерживаемые, в том числе и неиспользуемые, настройки программы.

### XSiteGen 1.3 — 2025-08-03

- **Добавлено:** так называемые "специальные" переменные, которые могут применяться к исходным файлам и каталогам.
- **Добавлено:** вывод пути к конфигурационному модулю после его успешной загрузки.
- **Обновлено:** название проекта.
- **Обновлено:** логотип.

### TinySSG 1.2 — 2025-07-15

Этот выпуск привносит новые возможности, с которыми пользоваться TinySSG стало удобнее.

- **Добавлено:** возможность загружать конфигурационный модуль (`config.py`) из любой папки, в которой был запущен TinySSG.
- **Добавлено:** рекурсивное сканирование исходных каталогов.
  
  Больше не нужно прописывать в конфигурационный модуль все подпапки внутри какой-либо папки — достаточно указать лишь саму папку.
  
- **Добавлено:** вывод количества файлов в исходных каталогах.
- **Обновлено:** логотип.
- **Удалено:**   проверка на наличие в исходном файле трёх и более строк текста (заголовка и одного абзаца текста).
- **Удалено:**   несколько бесполезных проверок и обработчиков исключений.

## TinySSG 1.1 — 2025-07-10

- **Добавлено:** маркеры в начале сообщений по их типам (информация, предупреждение, ошибка).
- **Обновлено:** логотип.
- **Обновлено:** документация.

## TinySSG 1.01 — 2025-06-23

- **Исправлено:** ошибка чтения файла, возникавшая при открытии файла шаблона без указания кодировки файла.

## TinySSG 1.0 — 2025-06-17

Первый публичный выпуск XSiteGen, тогда известного как "TinySSG".
