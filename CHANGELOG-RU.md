# История версий XSiteGen

#### [English](CHANGELOG.md) &bull; [Русский](CHANGELOG-RU.md).

### 1.3 *(2025-08-03)*

- Новое название проекта: XSiteGen.
- Добавлена поддержка "специальных" переменных, которые могут быть указаны для каждого исходного файла (или каталога с исходными файлами) по отдельности.
- После успешной загрузки конфигурационного модуля в консоль выводится путь к нему.

### 1.2 *(2025-07-15)*

Этот выпуск содержит новые интересные функции, направленные на улучшение удобства использования XSiteGen.

- Очередное обновление логотипа проекта.
- Появилась возможность загружать конфигурационный модуль (`config.py`) из рабочего каталога, в котором XSiteGen был запущен (в командной оболочке, к примеру).
- Реализовано рекурсивное сканирование каталогов с исходными файлами. Больше не нужно прописывать все подкаталоги в конфигурационном модуле — достаточно одного "корневого" каталога.
- Перед началом генерации веб-страниц выводится количество исходных файлов в каждом каталоге.
- Удалена бесполезная проверка каждого исходного файла на наличие трёх и более строк текста.
- Удалено несколько бесполезных проверок и обработчиков исключений.
- Прочие исправления и улучшения.

### 1.1 *(2025-07-10)*

- Новый логотип проекта.
- Добавлены маркеры для улучшения оформления вывода программы и визуального разделения сообщений (предупреждения, ошибки, информативные сообщения).
- Переписана и улучшена документация (README).
- Прочие исправления и улучшения.

### 1.01 *(2025-06-23)*

Исправлена ошибка чтения файла, которая могла возникнуть при открытии файла шаблона страницы без указания кодировки файла.

### 1.0 *(2025-06-17)*

Первый публичный выпуск XSiteGen, тогда известного под названием "TinySSG".
